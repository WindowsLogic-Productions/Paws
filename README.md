## Paws
Paws is a lightweight Kemono-style website client for Windows 7, 8.1, 10 & 11 using WebView2 and C#.

## Planned Features
- Run at startup.
- Hardware acceleration toggle.
- Open links in default browser toggle.
- Clear cache & data.
- User scripts (JavaScript injection).

## Want to help?
You'll need Visual Studio 2022 or 2026 with .NET Desktop Development to contribute to the project.
